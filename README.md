# Obsidian Project Automation

Плагин для автоматизации проектов и задач в Obsidian.

## Установка

1. Соберите плагин: `npm install && npm run build`.  
   Сборка создаёт `main.js` и `styles.css`. В каталоге плагина в Obsidian нужны **main.js**, **manifest.json**, **styles.css** и **defaults.json**.
2. Скопируйте каталог с плагинов внутрь Obsidian каталога .obsidian/plugins.
3. Включите плагин в Obsidian: Settings (ctrl+,) → Community plugins → Obsidian Project Automation.
4. В настройках плагина включите нужные модули и опции.

## Модули

- **Доска задач** - блок кода `opa-home-view`. Выводит таблицу всех активных задач, сгруппированных статусам. Позволяет менять статус задачи прямо из таблицы.
- **Напоминания** - блок кода `opa-reminders-view`. Умный список напоминаний. Поддерживает теги дат `(@DD-MM-YYYY HH:mm)` и повторения `(every 1 week)`. Умеет присылать всплывающие уведомления внутри Obsidian с возможностью отложить на час/завтра/дату.
- **Блокнот** - блок кода `opa-inbox-view`. Место для быстрого сброса мыслей. Любую запись можно отметить как выполненную, удалить, превратить в задачу или напоминание. Есть архив выполненного.
- **Активности** - блок кода `opa-activities-view`. Список ежедневных дел, для которых не нужны напоминания. Ведется статистика с графиками по дням.
- **Геймификация** - блок кода `opa-gamification-view`. Дашборд игрового прогресса. Выполнение задач, активностей и напоминаний дает XP и Gold, которые можно тратить во встроенном "Магазине" (который в свою очередь можно настроить под себя).

# Горячие клавиши (Hotkeys)

Плагин добавляет ряд команд. 
Рекомендуется назначить на них горячие клавиши в настройках Obsidian (Hotkeys):

-   **Создать задачу или проект** - Универсальное всплывающее окно для быстрого создания новой задачи или проекта.
-   **Создать задачу** - Открывает форму создания задачи. Позволяет выбрать шаблон, проект, контекст, окружение, сложность и дату планирования.
-   **Создать проект** - Создает новую заметку проекта по шаблону.
-   **Создать ежедневную заметку** - Создает заметку на текущий день в папке `periodic/daily/`.
-   **Открыть или создать домашнюю страницу** - Открывает файл `Homepage.md` (или создает его, наполняя всеми модулями плагина по умолчанию).

# Workflow

1. **Задача** - это отдельная заметка. У неё есть свойства (Frontmatter): статус, проект, контекст и прочие.
2. **Ежедневная заметка**. Вы не пишете прогресс в самой задаче. Вы открываете (или создаете) ежедневную заметку, добавляете заголовок ссылку на задачу `### [[Название задачи]]` и пишете прогресс под ним. Внутри заметки с задачей используется блок `opa-task-view`, который автоматически собирает и показывает всё, что вы писали об этой задаче в разные дни в ежедневных заметках.
3. **Проект**. Задачи агрегируются в проекты.
